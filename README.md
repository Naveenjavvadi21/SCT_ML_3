---

## 📖 Project Overview

This project was developed as part of the **SkillCraft Technology Machine Learning Internship – Task 3**. The objective is to build an image classification model that distinguishes between **cats** and **dogs** using the **Support Vector Machine (SVM)** algorithm.

The project covers the complete machine learning pipeline, including image preprocessing, feature extraction, feature scaling, model training, evaluation, and prediction.

---

## 🎯 Objectives

- Classify cat and dog images using SVM.
- Preprocess image data for machine learning.
- Train an SVM classifier with the RBF kernel.
- Evaluate model performance using standard classification metrics.
- Visualize predictions and confusion matrix.

---

## 📂 Dataset

**Dataset:** Dogs vs Cats Dataset

The dataset contains labeled images of:

- 🐱 Cats
- 🐶 Dogs

Images are resized and converted into numerical feature vectors before training.
